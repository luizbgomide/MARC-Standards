FIELD DEFINITION AND SCOPE

Classification number assigned using a scheme other than the the one required for field 050 (Library of Congress Classification), 060 (National Library of Medicine Classification Number), 070 (National Agricultural Library Classification Number), 080 (Universal Decimal Classification Number), or 082 (Dewey Decimal Classification Number). The scheme must be identified in subfield $2.

Contains numbers assigned from foreign classification schemes as well as from U.S. classification schemes. The field is not to be used for classification numbers assigned from a source for which a subfield $2 source of number code would not be assigned.

GUIDELINES FOR APPLYING CONTENT DESIGNATORS

■ INDICATORS

  • Both indicator positions are undefined; each contains a blank (#).

■ SUBFIELD CODES

$a - Classification number
$2 - Number source
MARC code for the source of the classification system or scheme from which the number was assigned. Code from: Classification Scheme Source Codes.
$6 - Linkage
See the description of this subfield in Appendix A: Control Subfields.
$8 - Field link and sequence number
See the description of this subfield in Appendix A: Control Subfields.

INPUT CONVENTIONS

Punctuation - Field 084 does not end with a period.
Capitalization - Capitalization of alphabetic characters in the classification number should follow the rules of the scheme. For most schemes, characters are recorded in uppercase.
