FIELD DEFINITION AND SCOPE

Classification number or call number assigned from Library of Congress Classification or LC Classification--Additions and Changes, both of which are maintained by the Library of Congress.

The second indicator values distinguish between content actually assigned by the Library of Congress and content assigned by an organization other than LC.

GUIDELINES FOR APPLYING CONTENT DESIGNATORS

■ INDICATORS

First Indicator - Undefined
Undefined and contains a blank (#).
Second Indicator - Source of class number
Value that indicates whether the number has been assigned by the Library of Congress or another organization.
0 - Assigned by LC
Source of the class number is the Library of Congress. Also used when an institution is transcribing field 050 from LC cataloging copy.
4 - Assigned by agency other than LC
Source of the class number is an organization other than the Library of Congress.
110 2#$aHospice of Huntington, Inc.
050 #4$aR726.8

■ SUBFIELD CODES

$a - Classification number
The source of the classification number is Library of Congress Classification and the LC Classification-Additions and Changes that are maintained by the Library of Congress. Subfield $a is repeated to record an alternative class number.
110 2#$aAmerican Dance Therapy Association.
050 #4$aRC489.D3
110 2#$aAutism Training Center.
050 #4$aRJ506.A9
$b - Item number
Item number portion of a call number. An item number is the Cutter, date, term, etc. that is added to a classification number to distinguish an entity from any other entity assigned the same classification number. The delimiter and subfield code b precede the last capital letter in the call number or the period, if present, preceding it.
Organizations that use the Cutter-Sanborn Three-Figure Author Table may conform to LC item number practice by applying Subject Cataloging Manual: Shelflisting procedures.
$3 - Materials specified
Part of the described material to which the field applies.
Used with archival-type materials; its use is parallel with the use of subfield $3 in other fields.
$6 - Linkage
See the description of this subfield in Appendix A: Control Subfields.
$8 - Field link and sequence number
See the description of this subfield in Appendix A: Control Subfields.

INPUT CONVENTIONS

Punctuation - Field 050 does not end with a period.
Capitalization - Alphabetic characters in the classification number are generally uppercase.
Spacing - Any spaces that are desired as part of the classification number must be input.
