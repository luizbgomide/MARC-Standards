FIELD DEFINITION AND SCOPE

Code that identifies the type of program or organization involved; for instance, whether it is a nonprofit, private, government, religious, charitable entity.

GUIDELINES FOR APPLYING CONTENT DESIGNATORS

■ INDICATORS

  • Both indicator positions are undefined; each contains a blank (#).
