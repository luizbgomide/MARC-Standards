040 - Record Source (NR)

MARC 21 Community Information - Concise
2007

MARC code or the name of the organization(s) that created the original record, assigned MARC content designation and transcribed the record into machine-readable form, or modified an existing MARC record. See Organization Code Sources for a listing of organization code sources used in MARC 21 records.

Indicators
First - Undefined
# - Undefined
Second - Undefined
# - Undefined
Subfield Codes
$a - Originating agency (R)
$b - Language code (NR)
MARC code for the language used in the textual portion of the record in the Note fields (5XX). Code from: MARC Code List for Languages.
$c - Transcribing agency (NR)
$d - Modifying agency (R)
$6 - Linkage (NR)
See the description of this subfield in Appendix A: Control Subfields.
$8 - Field link and sequence number (R)
See the description of this subfield in Appendix A: Control Subfields.
Examples
040 ##$aCP$cCP
040 ##$aNational Health Information Center$cNational Health Information Center
040 ##$aNN$bspa$cNN
040 ##$aOkT$cOkT
