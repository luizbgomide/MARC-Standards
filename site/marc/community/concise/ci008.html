008 - Fixed-Length Data Elements-General Information (NR)

MARC 21 Community Information - Concise
2007

Fifteen character positions (00-14) that contain positionally-defined, coded information about the record as a whole or about special community information aspects of the item being recorded. Each character position must contain either a defined code or a fill character ( | ). Code n (Not applicable), when it is defined for a data element, indicates that the character position definition does not apply.

Character Positions
00-05 - Date entered on file
Computer-generated, six-character numeric stringthat indicates the date the MARC record was created. The date is recorded in the pattern yymmdd.
06 - Volunteer opportunities
Whether opportunities for volunteers are available. This information may be described textually in field 571 (Volunteers Note),first indicator value 0 (Opportunities).
a - No volunteer opportunities
b - Volunteer opportunities
n - Not applicable
u - Unknown
07 - Volunteers provided
Whether the services of volunteers are provided. This information may be described textually in field 571 (Volunteers Note), first indicator value 1 (Services provided).
a - No volunteers provided
b - Volunteers provided
n - Not applicable
u - Unknown
08 - Child care arrangements
Whether child care is provided. This information may be described textually in field 576 (Services Available Note), first indicator value 0 (Child care arrangements).
a - No child care arrangements
b - Child care arrangements
n - Not applicable
u - Unknown
09 - Speakers bureau
Whether speakers are available. This information may be described textually in field 576 (Services Available Note), first indicator value 1 (Speakers bureau).
a - No speakers bureau
b - Speakers bureau
n - Not applicable
u - Unknown
10 - Mutual support groups
Whether meetings of informal groups to discuss specific concerns are available. This information may be described textually in field 576 (Services Available Note), first indicator value 2 (Mutual support groups).
a - No mutual support groups
b - Mutual support groups
n - Not applicable
u - Unknown
11 - Meeting rooms and facilities available
Whether rooms or other facilities are available. This information may be described textually in field 311 (Meeting Rooms and Facilities Available).
a - No meeting rooms and facilities
b - Meeting rooms and facilities
n - Not applicable
u - Unknown
12-14 - Language
Three-character MARC code that indicates the language associated with the community information entity. Code from: MARC Code List for Languages. When more than one language is associated with the entity, all appropriate codes are given in field 041 (Language Code) and the predominant language code is recorded in 008/12-14. Language information may be described textually in field 546 (Language Note).
