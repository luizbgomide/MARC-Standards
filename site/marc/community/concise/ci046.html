046 - Special Coded Dates (R)

MARC 21 Community Information - Concise
2007

Coded date information that is potentially useful for retrieval and data management purposes. If none of these types of dates is applicable to the record in hand, the field is not used.

Indicators
First - Undefined
# - Undefined
Second - Undefined
# - Undefined
Subfield Codes
$f - Action date (NR)
Date the record needs to be looked at again.
$g - Purge date (NR)
Date the information should be removed from the system as it is no longer relevant.
$h - Beginning date of event or program (NR)
$i - Ending date of event or program (NR)
$j - Date resource modified (NR)
$k - Beginning or single date created (NR)
$l - Ending date created (NR)
$m - Beginning of date valid (NR)
$n - End of date valid (NR)
$2 - Source of date (NR)
Code that identifies the source of the date scheme used in the field. Code from: Date and Time Scheme Source Codes.
$6 - Linkage (NR)
See the description of this subfield in Appendix A: Control Subfields.
$8 - Field link and sequence number (R)
See the description of this subfield in Appendix A: Control Subfields.
Examples
046 ##$f19931205
046 ##$f199312uu
046 ##$g19931010$h19931006$i19931009
046 ##$f19930904
046 ##$m20011008$n20011027
