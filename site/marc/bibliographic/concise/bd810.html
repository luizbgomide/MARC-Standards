810 - Series Added Entry-Corporate Name (R)

MARC 21 Bibliographic - concise
July 2022

Author/title series added entry in which the author portion is a corporate name.

Indicators

First - Type of corporate name entry element
0 - Inverted name
1 - Jurisdiction name
2 - Name in direct order
Second - Undefined
# - Undefined

Subfield Codes

$a - Corporate name or jurisdiction name as entry element (NR)
$b - Subordinate unit (R)
$c - Location of meeting (R)
$d - Date of meeting or treaty signing (R)
$e - Relator term (R)
$f - Date of a work (NR)
$g - Miscellaneous information (R)
$h - Medium (NR)
Media qualifier.

$p - Name of part/section of a work (R)

$s - Version (R)

$v - Volume/sequential designation (NR)
