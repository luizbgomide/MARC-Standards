256 - Computer File Characteristics (NR)

MARC 21 Bibliographic - concise
July 2022

Characteristics of a computer file, such as the type of file (e.g., Computer programs), the number of records, statements, etc. (e.g., 1250 records, 5076 bytes).

Indicators

First - Undefined
# - Undefined
Second - Undefined
# - Undefined

Subfield Codes

$a - Computer file characteristics (NR)
$6 - Linkage (NR)
See description of this subfield in Appendix A: Control Subfields.
$7 - Data provenance (R)
See description of this subfield in Appendix J: Data Provenance Subfields.
$8 - Field link and sequence number (R)
See description of this subfield in Appendix A: Control Subfields.

Examples

256 ##$aComputer data (2 files : 876,000, 775,000 records).
256 ##$aComputer programs (2 files : 4300, 1250 bytes).
256 ##$aData (1 file : 350 records).
