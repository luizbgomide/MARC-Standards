688 - Subject Added Entry-Type of Entity Unspecified (R)

MARC 21 Bibliographic - concise
November 2019

Subject added entry in which the entity type of the entry element is unspecified.

Indicators

First - Undefined
# - Undefined
Second - Source of name, title, or term
# - No information provided
7 - Source specified in subfield $2

Subfield Codes

$a - Name, title, or term (NR)
$e - Relator term (R)
$g - Miscellaneous information (R)
$0 - Authority record control number or standard number (R)
See description of this subfield in Appendix A: Control Subfields.
$1 - Real World Object URI (R)
See description of this subfield in Appendix A: Control Subfields.
$2 - Source of name, title, or term (NR)
MARC code that identifies the source list from which the subject added entry was assigned. It is used only when the second indicator position contains value 7 (Source specified in subfield $2). Code from: Subject Heading and Term Source Codes.
$3 - Materials specified (NR)
$4 - Relationship (R)
$6 - Linkage (NR)
See description of this subfield in Appendix A: Control Subfields.
$8 - Field link and sequence number (R)
See description of this subfield in Appendix A: Control Subfields.
