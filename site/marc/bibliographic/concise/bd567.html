567 - Methodology Note (R)

MARC 21 Bibliographic - concise
December 2017

Information concerning significant methodological characteristics of the material, such as the algorithm, universe description, sampling procedures, classification, or validation characteristics.

Indicators

First - Display constant controller
# - Methodology
8 - No display constant generated
Second - Undefined
# - Undefined

Subfield Codes

$a - Methodology note (NR)
$b - Controlled term (R)
Index term to describe the methodology used in the material is from a controlled subject heading system or thesaurus.
$0 - Authority record control number or standard number (R)
See description of this subfield in Appendix A: Control Subfields.
$1 - Real World Object URI (R)
See description of this subfield in Appendix A: Control Subfields.
$2 - Source of term (NR)
MARC code that identifies the source of the term used to record the methodology in subfield $b.

See description of this subfield in Appendix A: Control Subfields.
