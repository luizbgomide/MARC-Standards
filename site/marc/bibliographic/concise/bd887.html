887 - Non-MARC Information Field (R)

MARC 21 Bibliographic - Concise
2007

Data from non-MARC records for which there are no corresponding MARC 21 fields. Used when converting non-MARC records into the MARC 21 format.

Indicators
First - Undefined
# - Undefined
Second - Undefined
# - Undefined
Subfield Codes
$a - Content of non-MARC field (NR)
$2 - Source of data (NR)
Source of the data, either a schema or DTD reference. Code from: Format Source Codes.
Examples
887 ##$a<TextPublicationDate>20000617</TextPublicationDate>$2 [code for schema or DTD reference for ONIX]
887 ##$a<Box><eastlimit>0</eastlimit><westlimit>180</westlimit></Box>$2 [code for schema or DTD reference for DCMI-Box]
