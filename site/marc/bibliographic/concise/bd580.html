580 - Linking Entry Complexity Note (R)

MARC 21 Bibliographic - Concise
2007

Description of the complex relationship between the item described in the record and other items that cannot be adequately generated from the linking entry fields 760-787.

Indicators
First - Undefined
# - Undefined
Second - Undefined
# - Undefined
Subfield Codes
$a - Linking entry complexity note (NR)
Catalog entry for the related title and a statement describing the relationship.
$6 - Linkage (NR)
See description of this subfield in Appendix A: Control Subfields.
$8 - Field link and sequence number (R)
See description of this subfield in Appendix A: Control Subfields.
Examples
580 ##$aContinued in 1982 by: U.S. exports. Schedule E commodity groupings by world area and country.
580 ##$aMerged with: Index chemicus (Philadelphia, Pa. : 1977), to form: Current abstracts of chemistry and index chemicus (Philadelphia, Pa : 1978).
580 ##$aCumulates: Deutsche Bibliographie. Wöchentliches Verzeichnis.
580 ##$aContinued by: Ionospheric predictions issued by the laboratory under its later name: Institute for Telecommunication Sciences and Aeronomy.
