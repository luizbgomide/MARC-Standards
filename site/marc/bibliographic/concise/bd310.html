310 - Current Publication Frequency (R)

MARC 21 Bibliographic - concise
May 2020

Current stated publication frequency of either an item or an update to an item. Dates are included when the beginning date of the current frequency is not the same as the beginning date of publication.

Indicators

First - Undefined
# - Undefined
Second - Undefined
# - Undefined

Subfield Codes

$a - Current publication frequency (NR)
$b - Date of current publication frequency (NR)
Used when the date is different from the beginning date of the publication and a former publication frequency is given in field 321.
$0 - Authority record control number or standard number (NR)
See description of this subfield in Appendix A: Control Subfields.
$1 - Real World Object URI (R)
See description of this subfield in Appendix A: Control Subfields.
$2 - Source (NR)
$6 - Linkage (NR)
See description of this subfield in Appendix A: Control Subfields.
$8 - Field link and sequence number (R)
See description of this subfield in Appendix A: Control Subfields.
