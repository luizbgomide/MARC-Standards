385 - Audience Characteristics (R)

MARC 21 Bibliographic - concise
July 2022

A category of persons for which a resource is intended or a category of persons representing the intellectual level for which the content of a resource is considered appropriate.

Indicators

First - Undefined
# - Undefined
Second - Undefined
# - Undefined

Subfield Codes

$a - Audience term (R)
$b - Audience code (R)
$m - Demographic group term (NR)
$n - Demographic group code (NR)
$0 - Authority record control number or standard number (R)
See description of this subfield in Appendix A: Control Subfields.
$1 - Real World Object URI (R)
See description of this subfield in Appendix A: Control Subfields.
$2 - Source (NR)
Code from: Subject Heading and Term Source Codes or Target Audience Code and Term Source Codes.
$3 - Materials specified (NR)
$6 - Linkage (NR)
See description of this subfield in Appendix A: Control Subfields.
$7 - Data provenance (R)
See description of this subfield in Appendix J: Data Provenance Subfields.
$8 - Field link and sequence number (R)
See description of this subfield in Appendix A: Control Subfields.
