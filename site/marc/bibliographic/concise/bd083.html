083 - Additional Dewey Decimal Classification Number (R)

MARC 21 Bibliographic - concise
July 2022

Classification number used for subject access. It is taken from the Dewey Decimal Classification, accessed through electronic or printed versions.

Indicators

First - Type of edition
0 - Full edition
1 - Abridged edition
7 - Other edition specified in subfield $2
Second - Undefined
# - Undefined

Subfield Codes

$a - Classification number (R)
$c - Classification number--Ending number of span (R)
$m - Standard or optional designation (NR)
The following codes are used: a (standard) and b (optional).
$q - Assigning agency (NR)
Code from: MARC Code List for Organizations.
$y - Table sequence number for internal subarrangement or add table (R)
$z - Table identification (R)
$2 - Edition information (NR)
$6 - Linkage (NR)
See description of this subfield in Appendix A: Control Subfields.
$7 - Data provenance (R)
See description of this subfield in Appendix J: Data Provenance Subfields.
$8 - Field link and sequence number (R)
See description of this subfield in Appendix A: Control Subfields.

Examples

083 0#$z2$a4947$222
083 0#$a598.0994$222
083 0#$z2$a94$222
083 0#$a968.87/02$223/eng/20190402
