FIELD DEFINITION AND SCOPE

Subject added entry in which the entity type of the entry element is unspecified.

The field is used if no information is provided to identify the type of entity to which the subject heading belongs.

Subject headings which are not constructed by standard thesaurus-building conventions are recorded in field 653.

GUIDELINES FOR APPLYING CONTENT DESIGNATORS

■ INDICATORS

First Indicator - Undefined

Undefined and contains a blank (#).

Second Indicator - Source of name, title, or term

Subject heading system or thesaurus used in constructing the subject heading.

# - No information provided
7 - Source specified in subfield $2

Indicates that the source of the name, title, or term is contained in subfield $2.

■ SUBFIELD CODES

$a - Name, title, or term

Name, title, or term used as subject heading.

688 #7$aVenus$0(DE-2581)TH000006586$2gbd
688 #7$aForum Romanum$0(DE-2581)TH000010820$2gbd

$e - Relator term

Specifies the relationship between the heading and the described materials. More than one relationship term may be used if the entity has more than one relationship.

$g - Miscellaneous information

Data element that is not more appropriately contained in another defined subfield.

$0 - Authority record control number or standard number

See description of this subfield in Appendix A: Control Subfields.

688 #7$aImmanuel Kant (1724 - 1804)$0(DE-2581)TH000005362$2gbd
688 #7$aNaturrecht$0(DE-2581)TH000006673$2gbd

$1 - Real World Object URI

See description of this subfield in Appendix A: Control Subfields.

$2 - Source of name, title, or term

MARC code that identifies the source list from which the subject added entry was assigned. It is used only when the second indicator position contains value 7 (Source specified in subfield $2). Code from: Subject Heading and Term Source Codes.

688 #7$aStoiker$0(DE-2581)TH000006637$2gbd
688 #7$aRoma Urbs$0(DE-2581)TH000010811$2gbd

$3 - Materials specified

Part of the described materials to which the field applies.

$4 - Relationship

Code or URI that specifies the relationship from the entity described in the record to the entity referenced in the field. More than one relationship code or URI may be used if the entity has more than one relationship. A source of relationship codes is: MARC Code List for Relators.

$6 - Linkage

See description of this subfield in Appendix A: Control Subfields.

$8 - Field link and sequence number

See description of this subfield in Appendix A: Control Subfields.

INPUT CONVENTIONS

Punctuation - Field 688 does not end with a mark of punctuation or a closing parenthesis unless punctuation is part of the data.
