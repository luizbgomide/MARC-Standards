015 - National Bibliography Number (R)

MARC 21 LITE Bibliographic
April 2008

Bibliography entry number(s) for cataloging information that is derived from a national bibliography.

[Link to MARC 21 Concise Formats]

Indicators
First - Undefined
# - Undefined
Second - Undefined
# - Undefined
Subfield Codes
$a - National bibliography number (R)
$2 - Source (NR)
Code that identifies the source of the National Bibliography Number. Code from: MARC Code Lists for Relators, Sources, Description Conventions.
Examples
015 ##$aF84-3117
015 ##$aB67-20987$aB67-20988 (pbk)
015 ##$aGFR67-A14-54$aAu67-6
015 ##$aF67-835 (v. 1)$aF67-9455 (v. 2)
