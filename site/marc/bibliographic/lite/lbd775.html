775 - Other Edition Entry (R)

MARC 21 LITE Bibliographic
April 2008

Entry for another available edition of the target item (horizontal relationship). When a note is generated from this field, the introductory phrase Other editions available: may be generated based on the field tag for display.

[Link to MARC 21 Concise Formats]

Indicators
First - Note controller
0 - Display note
Second - Display constant controller
# - Other edition available
Subfield Codes

See the descriptions of the subfields under field 760.

$a - Main entry heading (NR)
$b - Edition (NR)
$d - Place, publisher, and date of publication (NR)
$e - Language code (NR)
Language code from field 008/35-37 (Language) of the related record.
$f - Country code (NR)
Country code from field 008/15-17 (Place of publication, production, or execution) of the related record.
$g - Relationship information (R)
$h - Physical description (NR)
$i - Display text (NR)
$n - Note (R)
$o - Other item identifier (R)
$t - Title (NR)
$w - Record control number (R)
$x - International Standard Serial Number (NR)
$z - International Standard Book Number (R)
