700 - Added Entry-Personal Name (R)

MARC 21 LITE Bibliographic
April 2008

Added entry in which the entry element is a personal name.

[Link to MARC 21 Concise Formats]

Indicators
First - Type of personal name entry element
0 - Forename
1 - Surname
3 - Family name
Second - Type of added entry
# - No information provided
Subfield Codes
$a - Personal name (NR)
$b - Numeration (NR)
$c - Titles and other words associated with a name (R)
$d - Dates associated with a name (NR)
$e - Relator term (R)
$q - Fuller form of name (NR)
$u - Affiliation (NR)
$4 - Relator code (R)
Examples
700 1#$aFord, John,$d1894-1973,$edirection.
700 0#$aFather Divine.
700 0#$aR. M. B.
700 1#$aVerez Peraza, Elena,$d1919-
