866 - Textual Holdings-Basic Bibliographic Unit (R)

MARC 21 LITE Bibliographic
April 2008

Textual description of the holdings of a basic bibliographic unit in the collections of the reporting organization.

[Link to MARC 21 Concise Formats]

Indicators
First - Field encoding level
# - No information provided
Second - Type of notation
0 - No information provided
1 - ANSI/NISO Z39.71 or ISO 10324
7 - Source specified in subfield $2
Subfield Codes
$a - Textual string (NR)
$x - Nonpublic note (R)
$z - Public note (R)
$2 - Source of notation (NR)
Examples
866 #1$80$a1-86 (1941-1987)$xbound in 2 v. per year$zSome issues missing
866 #1$a1974-1981$zSome issues lost
866 #1$av. 36-49 (1961-1974)$xincomplete vols. unbound$zsome issues missing
866 #1$av. 37-52$zBound; some issues missing
