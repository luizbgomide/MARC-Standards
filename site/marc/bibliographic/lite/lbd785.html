785 - Succeeding Entry (R)

MARC 21 LITE Bibliographic
April 2008

Information concerning the immediate successor to the target item (chronological relationship). When a note is generated from this field, the introductory phrase may be generated based on the value in the second indicator position for display.

[Link to MARC 21 Concise Formats]

Indicators
First - Note controller
0 - Display note
Second - Type of relationship
0 - Continued by
Subfield Codes

See the descriptions of the subfields under field 760.

$a - Main entry heading (NR)
$b - Edition (NR)
$d - Place, publisher, and date of publication (NR)
$g - Relationship information (R)
$h - Physical description (NR)
$i - Display text (NR)
$n - Note (R)
$o - Other item identifier (R)
$t - Title (NR)
$w - Record control number (R)
$x - International Standard Serial Number (NR)
$z - International Standard Book Number (R)
Example
785 00$tTEIC quarterly seismological bulletin$x0741-1898$w(DLC)sc#83007721#
