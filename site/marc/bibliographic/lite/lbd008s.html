008 - Continuing Resources (NR)

MARC 21 LITE Bibliographic
April 2008

[Link to MARC 21 Concise Formats]

Character Positions
18-20 - Supply fill characters (|)
21 - Type of continuing resource
One-character alphabetic code that indicates the type of continuing resource.
# - None of the following
Also used for yearbooks and annual reports.
d - Updating database
l - Updating loose-leaf
m - Monographic series
n - Newspaper
p - Periodical
w - Updating Web site
| - No attempt to code
22 - Supply fill character (|)
23 - Form of item
One-character alphabetic code that specifies the form of material for the item being described.
# - None of the following
a - Microfilm
b - Microfiche
c - Microopaque
d - Large print
f - Braille
r - Regular print reproduction
Eye-readable print, such as a photocopy.
s - Electronic
| - No attempt to code
24-34 - Supply fill characters (|)
