740 - Added Entry-Uncontrolled Related/Analytical Title (R)

MARC 21 LITE Bibliographic
April 2008

Added entries for related or analytical titles that are not controlled through an authority file or list. (If related or analytical titles are controlled by an authority file, use field 730 (Added entry - uniform title)).

[Link to MARC 21 Concise Formats]

Indicators
First - Nonfiling characters
0 - No nonfiling characters
1-9 - Number of nonfiling characters
Second - Type of added entry
# - No information provided
Subfield Code
$a - Uncontrolled related/analytical title (NR)
Example
740 0#$aManual del adivino.
