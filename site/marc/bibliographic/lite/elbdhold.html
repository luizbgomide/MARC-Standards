841-86X - Holdings, Location, Alternate Graphics Fields

MARC 21 LITE Bibliographic
April 2008

Fields 841-86X contain descriptions for data elements which are an integral part of the MARC 21 Format for Bibliographic Data and data elements which may appear either in bibliographic records or in separate MARC holdings records.

850 - Holding Institution (R)
852 - Location (R)
856 - Electronic Location and Access (R)
866 - Textual Holdings--Basic Bibliographic Unit (R)
867 - Textual Holdings--Supplementary Material (R)
868 - Textual Holdings--Indexes (R)
