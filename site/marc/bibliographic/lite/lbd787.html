787 - Nonspecific Relationship Entry (R)

MARC 21 LITE Bibliographic
April 2008

Information concerning the work related to the target item when the relationship does not fit any of those defined in fields 760-785.

[Link to MARC 21 Concise Formats]

Indicators
First - Note controller
0 - Display note
Second - Display constant controller
# - Related item
Subfield Codes

See the descriptions of the subfields under field 760.

$a - Main entry heading (NR)
$b - Edition (NR)
$d - Place, publisher, and date of publication (NR)
$g - Relationship information (R)
$h - Physical description (NR)
$i - Display text (NR)
$n - Note (R)
$o - Other item identifier (R)
$t - Title (NR)
$w - Record control number (R)
$x - International Standard Serial Number (NR)
$z - International Standard Book Number (R)
Examples
787 0#$aEmpire State report (1982)$x0747-0711$w(DLC)###84647292#$w(OCoLC)8541622
787 0#$tEmpire State report weekly$x0745-8622$w(DLC)###84647299#$w(OCoLC)9403759
