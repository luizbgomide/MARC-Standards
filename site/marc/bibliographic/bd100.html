FIELD DEFINITION AND SCOPE

Personal name used as a main entry in a bibliographic record.

Main entry is assigned according to various cataloging rules, usually to the person chiefly responsible for the work.

GUIDELINES FOR APPLYING CONTENT DESIGNATORS

Descriptions of the first indicator position and all subfield codes, as well as input conventions for the 100 field, are given in the X00 Personal Names-General Information section. Because the second indicator is different for various X00 fields, it is not described in the general information section, but is described below.

■ INDICATORS

Second Indicator - Undefined

Undefined and contains a blank (#).
