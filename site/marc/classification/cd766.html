FIELD DEFINITION AND SCOPE

Classification records where a secondary table applies or where the secondary table does not apply. Also provides a method of characterizing the entities in field 153 (Classification number) subfield $j (Caption) in the primary table so that a system can automatically link to the appropriate secondary table without first forcing the user to select from a menu.

GUIDELINES FOR APPLYING CONTENT DESIGNATORS

■ INDICATORS

  • Both indicator positions are undefined; each contains a blank (#).

■ SUBFIELD CODES

$a - Secondary table applicability
Code (e.g., code “a” for applicable; code “n” for not applicable) indicating whether or not a secondary table is applicable for breaking down the number or span of numbers in field 153 (Classification number).
$y - Type of division
Type of division to which the entity in field 153 (Classification number) subfield $j (Caption) belongs. The text matches that of the corresponding subfield $y (Type of division) in field 763 (Internal subarrangement or add table entry) in the schedule record with which the secondary table is used.
$6 - Linkage
See the description of this subfield in Appendix A: Control Subfields.
$8 - Field link and sequence number
See the description of this subfield in Appendix A: Control Subfields.
