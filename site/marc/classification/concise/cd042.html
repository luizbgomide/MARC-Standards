042 - Authentication Code (NR)

MARC 21 Classification - Concise
2007

One or more authentication codes assigned by centers that review and upgrade, if necessary, an authority record in a national database. Codes have not been defined.

Indicators
First - Undefined
# - Undefined
Second - Undefined
# - Undefined
Subfield Codes
$a - Authentication code (R)
