754 - Index Term-Faceted Topical Terms (R)

MARC 21 Classification - concise
December 2017

Topical subject constructed from a faceted vocabulary. For each term found in the field, an identification is given as to the facet/hierarchy in the thesaurus from which the term came. In addition, identification is given as to which term is the focus term of the expression

For descriptions of some subelements defined for this field, see the field 654 (Subject Added Entry--Uniform Title) section of the MARC 21 Concise Format for Bibliographic Data.

Indicators

First - Level of subject
# - No information provided
0 - No level specified
1 - Primary
2 - Secondary
Second - Undefined
# - Undefined

Subfield Codes

$a - Focus term (R)
$b - Non-focus term (R)
$c - Facet/hierarchy designation (R)
$v - Form subdivision (R)
$y - Chronological subdivision (R)
$z - Geographic subdivision (R)
$0 - Authority record control number or standard number (R)
See description of this subfield in Appendix A: Control Subfields.
$1 - Real World Object URI (R)
See description of this subfield in Appendix A: Control Subfields.
$2 - Source of heading or term (NR)
$3 - Material specified (NR)
$6 - Linkage (NR)
See the description of this subfield in Appendix A: Control Subfields.
$8 - Field link and sequence number (R)
See the description of this subfield in Appendix A: Control Subfields.
