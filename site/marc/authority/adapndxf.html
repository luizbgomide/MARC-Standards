Appendix F - Format Changes

MARC 21 Authority
December 2023

This section is provided for the lists of format changes that accompany each update to the MARC 21 Format for Authority Data.

MARC 21 FORMAT FOR AUTHORITY DATA

FORMAT CHANGE LIST

Update No. 37, December 2023
This page documents the changes contained in the thirty-seventh update to the 1999 edition of the MARC 21 Format for Authority Data. Update No. 37 (December 2023) includes changes resulting from proposals that were considered by the MARC 21 community since the publication of Update No. 36 (June 2023).

Format changes in previous updates can be found on the MARC Format Overview page.
UPDATE 37 CHANGES
■ New content designators:
Field
023Cluster ISSN (R)
Subfield

■ Obsolete content designators:
Subfields

■ Changes in content designator names and definitions:

UPDATE 37 LOCAL PRINT GUIDE

The following sections and fields were created/changed in Update No. 37 and should be printed to keep a print copy of the format up-to-date. Changes appear in red.

  • Summary Statement of Content Designators

  • 01X-09X - Numbers and Codes-General Information

  • 022 - International Standard Serial Number

  • 023 - Cluster ISSN

  • 371 - Address

  • 372 - Field of Activity

  • 374 - Occupation

  • 376 - Family Information
