384 - Key (NR)

MARC 21 Authority - concise
July 2022

The set of pitch relationships that establishes a single pitch class as a tonal centre for a musical work or expression. May be used to differentiate a musical work or expression from another with the same title.

Indicators

First - Key type  
# - Relationship to original unknown
0 - Original key
1 - Transposed key
2 - Key of representative expression
Second - Undefined
# - Undefined

Subfield Codes

$a - Key (NR)
$0 - Authority record control number or standard number (R)
See description of this subfield in Appendix A: Control Subfields.
$1 - Real World Object URI (R)
See description of this subfield in Appendix A: Control Subfields.
$6 - Linkage (NR)
See description of this subfield in Appendix A: Control Subfields.
$7 - Data provenance (R)
See description of this subfield in Appendix H: Data Provenance Subfields.
$8 - Field link and sequence number (R)
See description of this subfield in Appendix A: Control Subfields.

Examples

384 2#$aG major
384 0#$aD major
384 1#$aD major
