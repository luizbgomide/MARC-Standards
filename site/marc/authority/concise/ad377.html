377 - Associated Language (R)

MARC 21 Authority - concise
July 2022

Codes for languages associated with the entity described in the record. Includes the language a person uses when writing for publication, broadcasting, etc., a language a corporate body uses in its communications, a language of a family, or a language in which a work is expressed.

Indicators

First - Undefined
# - Undefined
Second - Source of code  
# - MARC language code

Code from: MARC Code List for Languages.

7 - Source specified in $2

Subfield Codes

$a - Language code (R)
$l - Language term (R)
$0 - Authority record control number or standard number (R)
See description of this subfield in Appendix A: Control Subfields.
$1 - Real World Object URI (R)
See description of this subfield in Appendix A: Control Subfields.
$2 - Source of code (NR)
Code from: Language Code and Term Source Codes.
$6 - Linkage (NR)
See description of this subfield in Appendix A: Control Subfields.
$7 - Data provenance (R)
See description of this subfield in Appendix H: Data Provenance Subfields.
$8 - Field link and sequence number (R)
See description of this subfield in Appendix A: Control Subfields.

Examples

377 ##$arus$aeng
377 ##$abnt$lLenje
377 ##$aeng
377 ##$anya$lChewa
377 ##$ahun$0http://id.loc.gov/vocabulary/languages/hun
377 ##$aeng$afre
377 #7$aen$afr$2iso639-1
