080 - Universal Decimal Classification Number (R)

MARC 21 Authority - concise
December 2017

Classification number associated with a 1XX heading in an established heading record. Number is taken from the Universal Decimal Classification. scheme.

Indicators

First - Type of edition
# - No information provided
0 - Full
1 - Abridged
Second - Undefined
# - Undefined

Subfield Codes

$a - Universal Decimal Classification number (NR)
$b - Item number (NR)
$x - Common auxiliary subdivision (R)
$0 - Authority record control number or standard number (R)
See description of this subfield in Appendix A: Control Subfields.
$1 - Real World Object URI (R)
See description of this subfield in Appendix A: Control Subfields.
$2 - Edition identifier (NR)
$6 - Linkage (NR)
See description of this subfield in Appendix A: Control Subfields.
$8 - Field link and sequence number (R)
See description of this subfield in Appendix A: Control Subfields.
