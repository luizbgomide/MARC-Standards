075 - Type of Entity (R)

MARC 21 Authority - concise
December 2017

Type of entity that is described by the authority record as a whole.

Indicators

First - Undefined
# - Undefined
Second - Undefined
# - Undefined

Subfield Codes

$a - Type of entity term (R)
$b - Type of entity code (R)
$0 - Authority record control number or standard number (R)
See description of this subfield in Appendix A: Control Subfields.
$1 - Real World Object URI (R)
See description of this subfield in Appendix A: Control Subfields.
$2 - Source (NR)
Code from: Type of Entity Source Codes.

Examples

075 ##$aPerson$bp$2gndgen
075 ##$aGott$bpxg$2gndspec
075 ##$aPerson$bp$2gndgen
075 ##$aFamilie$bpif$aMitglied regierender Fürstenhäuser$bpik$2gndspec
075 ##$aKörperschaft$bb$2gndgen
075 ##$aProjekt$bkip$2gndspec
075 ##$aWerk$bu$2gndgen
075 ##$aWerk der Musik$bwim$aFassung eines Werks der Musik$bwif$2gndspec
