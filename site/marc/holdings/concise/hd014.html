014 - Linkage Number (R)

MARC 21 Holdings - Concise
2007

Control number, assigned by a bibliographic network, of a separate holdings record or its related bibliographic record for which a separate holdings record is made.

Indicators
First - Type of linkage number
0 - Holdings record number
1 - Bibliographic record number
Second - Undefined
# - Undefined
Subfield Codes
$a - Linkage number (NR)
$b - Source of number (NR)
MARC organization code for the network whose control number contained in subfield $a or $z. See Organization Code Sources for a listing of organization code sources used in MARC 21 records.
$z - Canceled or invalid linkage number (R)
$6 - Linkage (NR)
See description of this subfield in Appendix A: Control Subfields.
Examples
014 1#$a1605897$bOCoLC
014 0#$a0000-49030$z0000-39730$b[organization code]
