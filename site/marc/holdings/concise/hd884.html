884 - Description Conversion Information (R)

MARC 21 Holdings - concise
April 2015

Used to provide information about the origin of a MARC record which has been converted by machine from another metadata structure.

Indicators

First - Undefined
# - Undefined
Second - Undefined
# - Undefined

Subfield Codes

$a - Conversion process (NR)
$g - Conversion date (NR)
$k - Identifier of source metadata (NR)
$q - Conversion agency (NR)
Code from: MARC Code List for Organizations.
$u - Uniform Resource Identifier (R)
