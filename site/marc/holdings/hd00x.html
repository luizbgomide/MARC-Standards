DEFINITION AND SCOPE

Variable control fields contain record control numbers and other kinds of control and coded information that are used in the processing of separate MARC holdings records. These fields have no indicators or subfield codes.

Fixed-length fields contain various kinds of information and have data elements that are positionally defined. A more detailed explanation of positionally defined data elements is found in the sections describing specific fields.
