DEFINITION AND SCOPE

Fields 010-066 contain standard numbers that identify the related MARC bibliographic record to which a separate holdings record is linked. Field 035 contains a system control number of a system other than the one whose separate holdings record control number is contained in field 001 (Control Number) or whose bibliographic record control number is contained in field 004 (Control Number for Related Bibliographic Record). Field 066 contains information specifying the character sets present in the record.
