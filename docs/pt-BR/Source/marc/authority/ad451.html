FIELD DEFINITION AND SCOPE

Tracing for a geographic name see from reference.

Used in an established heading or established heading and subdivision record to trace a see from reference from a geographic name not used in an established heading.

GUIDELINES FOR APPLYING CONTENT DESIGNATORS

Most of the content designators defined for field 451 are the same as those defined for field 151 (Heading-Geographic Name), field 551 (See Also From Tracing-Geographic Name), and field 751 (Established Heading Linking Entry-Geographic Name). Description, guidelines, and input conventions for shared content designators are given in the X51 Geographic Names-General Information section.

Guidelines for using subfield $w (Control subfield), subfield $i (Relationship information), and subfield $4 (Relationship) in field 451 are given in the Tracings and References-General Information section.
