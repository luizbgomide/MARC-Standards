1999 Edition
Update No. 1 (October 2000) through Update No. 26 (April 2018)

This online publication provides access to both the full and concise versions of the MARC 21 Format for Authority Data. The "full" authority format contains detailed descriptions of every data element, along with examples, input conventions, and history sections. The "concise" authority format contains abridged descriptions of every data element, along with examples. The full and concise versions are identified in the header of each field description.

Changes to the MARC 21 Format for Authority Data that resulted from Update No. 26 (April 2018) are displayed in red print. The date located in the header of the full version of each field indicates the last month and year of update.
