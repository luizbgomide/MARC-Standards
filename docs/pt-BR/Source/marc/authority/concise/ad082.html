082 - Dewey Decimal Call Number (R)

MARC 21 Authority - concise
June 2021

Call number for a series when all or part of the series is classified as a collected set or with the main series (646, Series Classification Practice, subfield $a, code c or m). Source of the classification number is the Dewey Decimal Classification, accessed through electronic or printed versions.

Indicators

First - Type of edition
0 - Full
1 - Abridged
7 - Other edition specified in subfield $2
Second - Source of call number
# - No information provided
0 - Assigned by LC
4 - Assigned by agency other than LC

Subfield Codes

$a - Classification number (NR)
$b - Item number (NR)
$d - Volumes/dates to which call number applies (NR)
Statement that is used only when the call number does not apply to all items in the series. Applies to: may be generated for display.
$2 - Edition information (NR)
$5 - Institution to which field applies (R)
See description of this subfield in Appendix A: Control Subfields.
$6 - Linkage (NR)
See description of this subfield in Appendix A: Control Subfields.
$8 - Field link and sequence number (R)
See description of this subfield in Appendix A: Control Subfields.
