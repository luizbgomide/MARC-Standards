110 - Primary Name-Corporate (NR)

MARC 21 Community Information - concise
December 2017

Corporate name used as the primary name associated with the record.

Indicators

First - Type of corporate name entry element
0 - Inverted name
1 - Jurisdiction name
2 - Name in direct order
Second - Undefined
# - Undefined

Subfield Codes

$a - Corporate name or jurisdiction name as entry element (NR)
Name of a corporate body or the first entity when subordinate units are present; a jurisdiction name under which a corporate body, city section, or title of a work is entered; or a jurisdiction name that is also an ecclesiastical entity.
$b - Subordinate unit (R)
Name of a subordinate corporate unit, city section, or meeting entered under a corporate or jurisdiction name.
$c - Location of meeting (R)
Place name or a name of an institution where a meeting was held.
$d - Date of meeting (NR)
$e - Relator term (R)
$g - Miscellaneous information (R)
Subelement that is not more appropriately contained in subfield $c, $d, or $n in a meeting entered under a corporate name; or information that is not more appropriately contained in another defined subfield in any other type of corporate name field.
$n - Number of meeting (NR)
Number designation for a meeting entered under a corporate name.
$u - Affiliation (NR)
Affiliation or address of the name.
$0 - Authority record control number or standard number (R)
See description of this subfield in Appendix A: Control Subfields.
$1 - Real World Object URI (R)
See description of this subfield in Appendix A: Control Subfields.
$4 - Relator code (R)
MARC code for the relator term that describes the relationship between a name and an entity. Code from: MARC Code List for Relators.
$6 - Linkage (NR)
See description of this subfield in Appendix A: Control Subfields.
$8 - Field link and sequence number (R)
See description of this subfield in Appendix A: Control Subfields.
