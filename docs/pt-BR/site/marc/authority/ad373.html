FIELD DEFINITION AND SCOPE

Information about a group, institution, association, etc., that is associated with the entity in the 1XX. This can also be a group with which a person is or has been affiliated, including dates of affiliation.

The field is repeated if the person has multiple affiliations for different time periods.

GUIDELINES FOR APPLYING CONTENT DESIGNATORS

■ INDICATORS

  • Both indicator positions are undefined; each contains a blank (#).

■ SUBFIELD CODES

$a - Associated group

A group that is associated with the entity in the 1XX or with which the entity is affiliated or has been affiliated through employment, membership, cultural identity, etc.

373 ##$aFaculty of Biological Science, Leeds University$s2000$t2005

$i - Relationship information

Designation of a relationship between the associated group recorded in the 373 field and the 1XX entity in the record. This may be an uncontrolled textual phrase or a controlled textual value from a list of relationships.

100 1#$aBrommel, Angela M.,$d1975-
373 ##$iGraduate of:$aAntioch University Los Angeles$4http://rdaregistry.info/Elements/a/P50253

$s - Start period

The starting date of the association.

$t - End period

The end date of the association.

$u - Uniform Resource Identifier

Uniform Resource Identifier (URI), for example a URL or URN, which provides electronic access data in a standard syntax. This data can be used for automated access to an electronic item using one of the Internet protocols.

In field 373, subfield $u is used to record the location of the source of information about associated groups available electronically.

$v - Source of information

Source where the information recorded in field 373 was found if not through a URI.

$0 - Authority record control number or standard number

See description of this subfield in Appendix A: Control Subfields.

$1 - Real World Object URI

$2 - Source of term

Specifies a code that identifies the source of a controlled vocabulary for affiliation terms in $a.

Code from: Name and Title Authority Source Codes.

$4 - Relationship

Code or URI that specifies the relationship from the entity described in the record to the entity referenced in the field.
