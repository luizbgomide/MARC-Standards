003 - Control Number Identifier (NR)

MARC 21 Holdings - Concise
2007

MARC code for the agency whose system control number is present in field 001 (Control Number).

See Organization Code Sources for a listing of organization code sources used in MARC 21 records.

Examples
003 DLC
003 OCoLC
003 DNLM
003 CaOONL
