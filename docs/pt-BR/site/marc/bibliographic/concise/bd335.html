335 - Extension Plan (R)

MARC 21 Bibliographic - concise
July 2022

A categorization reflecting an intention to extend the content of a work.

Indicators

First - Undefined
# - Undefined
Second - Undefined
# - Undefined

Subfield Codes

$a - Extension plan term (NR)
Extension plan of the work being described.
$b - Extension plan code (NR)
Code representing the extension plan of the work being described.
$0 - Authority record control number or standard number (R)
See description of this subfield in Appendix A: Control Subfields.
$1 - Real World Object URI (R)
See description of this subfield in Appendix A: Control Subfields.
$2 - Source (NR)
Code from: Genre/Form Code and Term Source Codes.
$3 - Materials specified (NR)
Part of the described materials to which the field applies.
$6 - Linkage (NR)
See description of this subfield in Appendix A: Control Subfields.
$7 - Data provenance (R)
See description of this subfield in Appendix J: Data Provenance Subfields.
$8 - Field link and sequence number (R)
See description of this subfield in Appendix A: Control Subfields.

Examples

335 ##$astatic plan$2rdaep
335 ##$asuccessive determinate plan$2rdaep
335 ##$asuccessive indeterminate plan$2rdaep
