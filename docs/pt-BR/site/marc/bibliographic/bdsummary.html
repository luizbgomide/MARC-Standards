RESUMO DOS DESIGNADORES DE CONTEÚDO

MARC 21 Bibliográfico
Dezembro de 2023
Documentação "completa" se refere ao Formato MARC 21 para Dados Bibliográficos que contém descrições detalhadas de cada elemento de dados, junto com exemplos, convenções de uso e seções de histórico.
Documentação "concisa" se refere ao Formato MARC 21 Conciso para Dados Bibliográficos que contém descrições resumidas de cada elemento de dados, junto com exemplos.
