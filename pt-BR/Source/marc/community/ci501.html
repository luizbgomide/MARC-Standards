501 - Currency of Information Note (R)

MARC 21 Community Information - Full
January 2000
First Indicator
Display constant controller
# - No information provided

Undefined

Subfield Codes
$a - Currency of information note (NR)

$8 - Field link and sequence number (R)

FIELD DEFINITION AND SCOPE

Note that specifies that the information pertaining to the community information entity is current as of a specific date.

GUIDELINES FOR APPLYING CONTENT DESIGNATORS

■ INDICATORS

First Indicator - Display constant controller
Value that controls the generation of a display constant preceding the note.
# - No information provided
When no information is provided, the display constant Information current as of: may be generated.
501 ##$aSeptember 10, 1993.
8 - No display constant generated
Second Indicator - Undefined
Undefined and contains a blank (#).

■ SUBFIELD CODES

$a - Currency of information note
Text of the note.
501 ##$a12/15/1992.
$6 - Linkage
See the description of this subfield in Appendix A: Control Subfields.
$8 - Field link and sequence number
See the description of this subfield in Appendix A: Control Subfields.

INPUT CONVENTIONS

Punctuation - Field 501 ends with a period unless another mark of punctuation is present.
Display Constant - Introductory phrase Information current as of: is not carried in the MARC record. It may be system generated as a display constant associated with the first indicator value.
First indicator Display constant
