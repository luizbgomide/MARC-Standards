"Concise" documentation refers to the MARC 21 Concise Format for Community Information that contains abridged descriptions of every data element, along with examples.

DEFINITION AND SCOPE

Record control number and other kinds of control and coded information that are used in the processing MARC community information records. Each control field is identified by a field tag in the Directory and contains either a single data element or a series of fixed-length data elements identified by relative character position. These fields have no indicators or subfield codes.

In both of the 007 Physical Description Fixed Field and the 008 Fixed-Length Data Elements Field, the specific data elements are positionally defined. A more detailed explanation of positionally defined data elements is found in the description of fields 007, and 008.
