FIELD DEFINITION AND SCOPE

Unique numbers that have been assigned to a record by a national bibliographic agency other than the Library of Congress.

Library of Congress control numbers are recorded in field 010. The control number may also appear in field 001 (Control Number) in records distributed by the same national agency. An organization using the record may remove the national bibliographic agency control number from field 001 and use field 001 for its own system control number.

A record may contain field 016 with a canceled or invalid control number of a previously-distributed record. A record may be canceled because it is a duplicate record for the same item.

GUIDELINES FOR APPLYING CONTENT DESIGNATORS

■ INDICATORS

First Indicator - National bibliographic agency
Value that either identifies a specific national bibliographic agency as the source of the control number, or refers to a code in subfield $2 (Source).
# - Library and Archives Canada
Library and Archives Canada (LAC) is the source of the national bibliographic agency control number.
7 - Source specified in subfield $2
Source of the control number is indicated by a code in subfield $2.
016 7#$a3945683$2[MARC code for agency]
Second Indicator - Undefined
Undefined and contains a blank (#).

■ SUBFIELD CODES

$a - Record control number
Valid record control number assigned by a national bibliographic agency.
016 7#$aLIS0003947$2[MARC code for agency]
$z - Canceled or invalid control number
Canceled or invalid control number assigned by a national bibliographic agency.
$2 - Source
MARC code or the name of the organization that identifies the national bibliographic agency that was the source of the record control number recorded in field 016. See MARC Code List for Organizations for a listing of organization code sources used in MARC 21 records.
Used only when the second indicator position contains value 7 (Source specified in subfield $2).
016 7#$a3296UL$z3948643a$2[MARC code for agency]
$8 - Field link and sequence number
See description of this subfield in Appendix A: Control Subfields.

CONTENT DESIGNATOR HISTORY

Field 016 - National Library of Canada Record Control Number [REDEFINED, 1997] [CAN/MARC only]
Both indicator positions were undefined. The defined subfield codes were $a (NLC record control number) and $z (Canceled/invalid NLC control number).
