Appendix F - Format Changes

MARC 21 Holdings
December 2023

This section is provided for the lists of format changes that accompany each update to the MARC 21 Format for Holdings Data.

MARC 21 FORMAT FOR HOLDINGS DATA

FORMAT CHANGE LIST

Update No. 37, December 2023
This page documents the changes contained in the thirty-seventh update to the 1999 edition of the MARC 21 Format for Holdings Data. Update No. 37 (December 2023) includes changes resulting from proposals that were considered by the MARC 21 community since the publication of Update No. 36 (June 2023).

Format changes in previous updates can be found on the MARC Format Overview page.
