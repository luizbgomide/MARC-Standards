"Full" documentation refers to the MARC 21 Format for Holdings Data that contains detailed descriptions of every data element, along with examples, input conventions and history sections.

"Concise" documentation refers to the MARC 21 Format for Holdings Data that contains abridged descriptions of every data element, along with examples.

  • 880 - Alternate Graphic Representation (R)  Full | Concise
  • 883 - Metadata Provenance (R)  Full | Concise
  • 884 - Description Conversion Information (R)  Full | Concise

FIELD DEFINITION AND SCOPE

Fields 880, 883, and 884 are the only 88X fields defined in the MARC 21 holdings format at the present time. For the most part, these fields carry over the definitions found in the other MARC 21 formats.
