610 - Subject Added Entry-Corporate Name (R)

MARC 21 LITE Bibliographic
April 2008

Subject added entry in which the entry element is a corporate name.

[Link to MARC 21 Concise Formats]

Indicators
First - Type of corporate name entry element
1 - Jurisdiction name
2 - Name in direct order
Second - Thesaurus
7 - Source specified in subfield $2
Subfield Codes
$a - Corporate name or jurisdiction name as entry element (NR)
$b - Subordinate unit (R)
$e - Relator term (R)
$t - Title of a work (NR)
$u - Affiliation (NR)
$v - Form subdivision (R)
$x - General subdivision (R)
$y - Chronological subdivision (R)
$z - Geographic subdivision (R)
$2 - Source of heading or term (NR)
Code from: MARC Code Lists for Relators, Sources, Description Conventions.
$4 - Relator code (R)
Examples
610 27$aCatholic Church.$bProvince of Baltimore (Md.)$2lcsh
610 27$aUnited Nations$zAfrica.$2lcsh
610 27$aTitanic (Steamship)$2lcsh
610 27$aEmpire State Building (New York, N.Y.)$2lcsh
610 17$aFrance.$tConstitution (1946)$2lcsh
610 27$aDaughters of the American Revolution.$bMary Tyler Chapter (Tyler, Tex.)$2lcsh
610 17$aUnited States.$bArmy.$bCavalry$xHistory$yCivil War, 1861-1865$vMaps.$2lcsh
