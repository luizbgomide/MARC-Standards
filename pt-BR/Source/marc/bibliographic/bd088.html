Subfield Codes
  • $a - Report number (NR)
  • $z - Canceled/invalid report number (R)
  • $6 - Linkage (NR)
  • $8 - Field link and sequence number (R)

FIELD DEFINITION AND SCOPE

Report number that is not a Standard Technical Report Number (STRN), recorded in field 027.

Field 088 is repeated for multiple report numbers, except when one number is the canceled or invalid number corresponding to a valid report number.

GUIDELINES FOR APPLYING CONTENT DESIGNATORS

■ INDICATORS

  • Both indicator positions are undefined; each contains a blank (#).

$z - Canceled/invalid report number

Each canceled or invalid report number is contained in a separate subfield $z. If no valid report number exists, only subfield $z should be recorded.

088 ##$aNASA-RP-1124-REV-3 $zNASA-RP-1124-REV-2
[One valid report number has been canceled in favor of another]
088 ##$zNASA-TN-D-8008
[Report number is invalid due to an error in assignment or printing on the item]

CONTENT DESIGNATOR HISTORY

Prior to 2015, series report numbers were included in fields 490/830, not in field 088.
