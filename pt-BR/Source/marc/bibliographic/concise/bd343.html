343 - Planar Coordinate Data (R)

MARC 21 Bibliographic - Concise
2007

Information about the coordinate system developed on a planar surface. The information is provided to allow the user of a geospatial data set to identify the quantities of distances, or distances and angles. These define the position of a point on a reference plane onto which the surface of the Earth has been projected.
