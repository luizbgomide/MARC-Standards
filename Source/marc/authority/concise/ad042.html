042 - Authentication Code (NR)

MARC 21 Authority - Concise
2007

One or more authentication codes indicating that the record, existing in a national database, has been reviewed in a specific way. The codes are associated with specifically designated authentication agencies.

Indicators
First - Undefined
# - Undefined
Second - Undefined
# - Undefined
Subfield Codes
$a - Authentication code (R)
If more than one code is applicable, the subfield is repeated. Code from: MARC Authentication Action Code List.
