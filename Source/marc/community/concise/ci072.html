072 - Human Services Code (R)

MARC 21 Community Information - Concise
2007

Human services code, i.e., a code that identifies the kinds of programs and/or services provided.

Indicators
First - Undefined
# - Undefined
Second - Code source
7 - Source specified in subfield $2
Subfield Codes
$a - Human services code (NR)
Code for the broad category of human services provided.
$x - Human services code subdivision (R)
Code that indicates the level of specificity within the broader category represented by the code in subfield $a.
$2 - Code source (NR)
MARC code that identifies the source list from which the human services code was assigned. Code from: Human Services Code Source Codes.
$6 - Linkage (NR)
See the description of this subfield in Appendix A: Control Subfields.
$8 - Field link and sequence number (R)
See the description of this subfield in Appendix A: Control Subfields.
