70X-75X - Added Entry Fields

MARC 21 LITE Bibliographic
April 2008

Fields 700-75X contain a name and/or title or a term that provides access to a bibliographic record that is not provided through main entry (1XX), subject access (6XX), series statement (4XX), series added entry (8XX), or title (20X-24X) fields.

700 - Added Entry--Personal Name (R)
710 - Added Entry--Corporate Name (R)
711 - Added Entry--Meeting Name (R)
720 - Added Entry--Uncontrolled Name (R)
730 - Added Entry--Uniform Title (R)
740 - Added Entry--Uncontrolled Related/Analytical Title (R)
752 - Added Entry--Hierarchical Place Name (R)
