082 - Dewey Decimal Classification Number (R)

MARC 21 Bibliographic - concise
July 2022

Classification number is taken from the Dewey Decimal Classification, accessed through electronic or printed versions.

Indicators

First - Type of edition
0 - Full edition
1 - Abridged edition
7 - Other edition specified in subfield $2
Second - Source of classification number
# - No information provided
0 - Assigned by LC

May be used by organizations transcribing from LC copy.

4 - Assigned by agency other than LC

Subfield Codes

$a - Classification number (R)
$b - Item number (NR)
$m - Standard or optional designation (NR)
The following codes are used: a (standard) and b (optional).
$q - Assigning agency (NR)
Code from: MARC Code List for Organizations.
$2 - Edition information (NR)
$6 - Linkage (NR)
See description of this subfield in Appendix A: Control Subfields.
$7 - Data provenance (R)
See description of this subfield in Appendix J: Data Provenance Subfields.
$8 - Field link and sequence number (R)
See description of this subfield in Appendix A: Control Subfields.
