088 - Report Number (R)

MARC 21 Bibliographic - concise
April 2015

Report number that is not a Standard Technical Report Number (STRN), recorded in field 027.

Indicators

First - Undefined
# - Undefined
Second - Undefined
# - Undefined

Subfield Codes

$a - Report number (NR)
$z - Canceled/invalid report number (R)
$6 - Linkage (NR)
See description of this subfield in Appendix A: Control Subfields.
$8 - Field link and sequence number (R)
See description of this subfield in Appendix A: Control Subfields.

Examples

088 ##$aSTRATLAB-71-98
088 ##$aAPA 3009
088 ##$aAPA 3010
088 ##$aAPA 3011
088 ##$aNASA-RP-1124-REV-3 $zNASA-RP-1124-REV-2
088 ##$zNASA-TN-D-8008
