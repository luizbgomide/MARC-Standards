758 - Resource Identifier (R)

MARC 21 Bibliographic - concise
May 2019

An identifier for a resource that is either the resource described in the bibliographic record or a resource to which it is related. Resources thus identified may include, but are not limited to, FRBR works, expressions, manifestations, and items. The field does not prescribe a particular content standard or data model.

Indicators

First - Undefined
# - Undefined
Second - Undefined
# - Undefined

Subfield Codes

$a - Label (NR)
$i - Relationship information (R)
$0 - Authority record control number or standard number (R)
See description of this subfield in Appendix A: Control Subfields.
$1 - Real World Object URI (R)
See description of this subfield in Appendix A: Control Subfields.
$2 - Source of heading or term (NR)
Code from: Name and Title Authority Source Codes.
$3 - Materials specified (NR)
$4 - Relationship (R)
$5 - Institution to which field applies (NR)
See description of this subfield in Appendix A: Control Subfields.
$6 - Linkage (NR)
See description of this subfield in Appendix A: Control Subfields.
$8 - Field link and sequence number (R)
See description of this subfield in Appendix A: Control Subfields.
