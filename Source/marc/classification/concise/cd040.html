040 - Record Source (NR)

MARC 21 Classification - Concise
2007

MARC code or name of the organization(s) that created the original classification record, assigned MARC content designation and transcribed the record into machine-readable form, or modified an existing MARC 21 record. See Organization Code Sources for a listing of sources used in MARC 21 records.

Indicators
First - Undefined
# - Undefined
Second - Undefined
# - Undefined
Subfield Codes
$a - Original cataloging agency (NR)
$b - Language of cataloging (NR)
Code from: MARC Code List for Languages.
$c - Transcribing agency (NR)
$d - Modifying agency (R)
$6 - Linkage (NR)
See the description of this subfield in Appendix A: Control Subfields.
$8 - Field link and sequence number (R)
See the description of this subfield in Appendix A: Control Subfields.
Examples
040 ##$aDLC$cDLC
040 ##$aIU$cIU
040 ##$aNLM$cNLM
